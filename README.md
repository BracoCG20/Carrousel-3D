# Carrusel 3D con CSS

Un componente de carrusel 3D interactivo, creado principalmente con HTML y CSS. Ideal para galerías de imágenes o para destacar contenido de una manera visualmente atractiva.

## Descripción

Este proyecto es una demostración de cómo crear un carrusel de imágenes con un efecto de perspectiva 3D utilizando transformaciones y animaciones de CSS3. El objetivo es proporcionar un componente ligero y fácil de integrar que no dependa de librerías pesadas de JavaScript para su funcionalidad principal.

## Características

- **Efecto 3D:** Los elementos giran en un espacio tridimensional creando una experiencia inmersiva.
- **Animado con CSS:** La animación y el efecto principal se logran con CSS, haciéndolo muy eficiente.
- **Fácil de Personalizar:** Puedes cambiar las imágenes y el contenido modificando simplemente el HTML.

## Tecnologías Utilizadas

Este proyecto fue construido utilizando tecnologías web estándar:

- HTML5
- CSS3
